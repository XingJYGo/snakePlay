### **JavaScript 经典面向对象Demo-贪吃蛇**

![snake](/snake.gif)

​    面向对象编程思想在程序项目中有着非常明显的优势:

  1 代码可读性高.由于继承的存在，即使改变需求，那么维护也只是在局部模块

   2 维护非常方便并且成本较低。

​     这个demo是采用了面向对象的编程思想. 用JavaScript 语言编写的游戏小程序--贪吃蛇.

​    代码注释详细,逻辑清晰 . 非常适合新手前端开发者, 锻炼JavaScript语言的面向对象的编程思想.  欢迎大家下载.
